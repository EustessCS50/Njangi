Njangi — API Help

About

Njangi is a minimal Django application that provides membership and savings group management features. The app supports:

  • Members and memberships
  • Meetings and contributions
  • Loans and repayments
  • Expenses tracking
  • Bank accounts and transactions

Developer: Besong E. Besong

API Base

The API root for this installation is at /api/. The available resources include:

  • /api/groups/ — manage Njangi groups (accounts)
  • /api/members/
  • /api/memberships/
  • /api/meetings/
  • /api/contributions/
  • /api/loans/
  • /api/repayments/
  • /api/expenses/
  • /api/bank/accounts/
  • /api/bank/transactions/

Authentication

The API uses JWT authentication. Obtain a token with:

POST /api/token/ {"username": "", "password": ""}

Use the returned access token in the Authorization header for requests:

Authorization: Bearer <access_token>

Refresh tokens at /api/token/refresh/.

Common Usage Examples

List members (scoped to groups via the objects' group field; filter by group id or use Group endpoints):

curl -sS -H "Authorization: Bearer <token>" https://your-host.example/api/members/

Create a member (JSON):

curl -X POST -H "Content-Type: application/json" -H "Authorization: Bearer <token>" -d '{"first_name":"Jane","last_name":"Doe"}' https://your-host.example/api/members/
